Logistic Regression

Logistic Regression applies the sigmoid function to produce binary outputs — Safe (0) or Malicious (1).

σ(z) = 1 / (1 + e−z)
z = β₀ + β₁x₁ + β₂x₂ + β₃x₃ + β₄x₄

Features (x₁ … x₄):

  • x₁ URL Length
  • x₂ HTTPS Present (0 / 1)
  • x₃ Special Character Count
  • x₄ IP-based URL (0 / 1)
σ(z) ≤ 0.5 → Safe
σ(z) > 0.5 → Malicious
